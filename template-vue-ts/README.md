## 安装
```bash
# 安装依赖
pnpm install
# 启动服务
pnpm dev
```
## 开发
```bash
# 开发模式
pnpm dev
# 构建生产环境
pnpm build
# 预览生产环境
pnpm preview
```
## 部署

```bash
# 部署到服务器
pnpm deploy
```

> 注意：部署前请先将 `.env.production` 文件中的 `VITE_APP_BASE_API` 配置为你的服务器地址。请先进行对本地部署环境对部署服务器进行 `ssh` 密钥验证。
